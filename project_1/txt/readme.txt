Описание проекта

	1. Данный проект предназначен для формирвания предматчевой статистики двух встречающихся команд

	2. Статистика выводится в красивом графическом оформленном стиле

	3. Исходные данные для статистики получаются в результате парсинга и формирования двух json файлов



Формирование массива с данными команд (парсинг таблиц конференций)

	файл - project_1/result_teams.php
	1. в папку table_conf сохраняются актуальные данные таблиц конференций - https://www.championat.com/hockey/_superleague/2202/table/all.html
		west_17092017 (конференция, дата сохранения)
		east_17092017
		
	2. в результате парсинга этих двух файлов получается массив team[], содержащий данные
		team['сокращенное название команды']['games'] - количество проведенных игр
											['clear_wins'] - количество чистых побед
											['ot_wins'] - побед в овертайие
											['b_wins'] - побед по буллитам
											и т.д.
	
	3.	вызывами функции name_team() массив team[] кодируется в json, после чего
		json файлы записываются в папку json 
		east_24_09_2017.json
		west_24_09_2017.json
